# Used Technologies
💅 FrontEnd : Js, Css, Bootstrap

🛠 Backend : Laravel 8

💾 Database : MySql

☁️ Deployment : Github

To run this project make sure you ran these commands

composer install

copy .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve

Now you can visit localhost:8000 and see project
